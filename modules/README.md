# Modules

Each subdirectory is a **module** with the same layout as the repository root (`bom/`, `cad/`, `architecture/`, `docs/`, `manufacturing/`, `modules/`).

- **Embedded (default):** plain folders here.
- **Extracted:** same path as a Git submodule — links and imports unchanged.

Adapters for interface version bridges live under `modules/adapters/` by convention.
