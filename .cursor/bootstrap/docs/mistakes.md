# Mistake log

Log errors, wrong approaches, and prevention rules so agents and humans do not repeat them. For repeated issues, promote prevention rules into scoped `.cursor/rules/`; keep the full incident history here.

## Format (copy per incident)

```markdown
## [Short description] — [Date]
**What happened:** [One or two sentences]
**Root cause:** [Why did this happen?]
**Fix applied:** [What was done to correct it?]
**Prevention rule:** [One concrete rule to avoid this in the future]
**Affected files:** [List of files involved]
```
